## Metric Definitions

**Return (1d)** - One-day performance return

**Return (30d)** - 30-day performance return

**Return (1Y)** - 12-month performance return

**CAGR** - Compound Annual Growth Rate; average annual return accounting for compounding since inception

**Max DD** - Maximum Drawdown; worst peak-to-trough decline experienced during backtest

**Beta** - Market sensitivity relative to S&P 500 (1.0 = market, >1.0 = more volatile, <1.0 = less volatile)

**Alpha** - Risk-adjusted outperformance vs. benchmark; positive = beating market on risk-adjusted basis

**Sharpe** - Sharpe Ratio; risk-adjusted return (return per unit of risk taken); higher is better

**Win Rate** - Percentage of profitable trades

**Avg Win** - Average return on winning trades

**Avg Loss** - Average loss on losing trades

**Volatility** - Annual Volatility; dispersion of daily returns

**Info Ratio** - Information Ratio; excess return per unit of tracking error; measures skill vs. benchmark

**Treynor** - Treynor Ratio; return per unit of systematic (market) risk

**Trades** - Total number of trades executed during backtest period
